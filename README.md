# NextJS Template

> By: Jehu Zubiri

> ---

> Setup Guide:

1. After cloning, run these on terminal
2. npm i
3. npm i next
4. npm run build
5. npm i -g env-cmd
6. npm run development | staging | production

> Modules:

1. @reduxjs/toolkit
2. react-redux
3. nextjs-progressbar
4. sass
5. axios
6. js-cookie
7. lodash
